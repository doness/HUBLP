# Aika Nexus - Astro Migration

This project has been migrated to [Astro](https://astro.build/).

## Prerequisites

- Node.js (v18.14.1 or higher)
- npm (comes with Node.js)

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

4.  **Preview Production Build**:
    ```bash
    npm run preview
    ```

## Project Structure

- `src/pages`: Website pages (`index.astro`, `privacy.astro`, `terms.astro`).
- `src/components`: UI components (`Header.astro`, `Footer.astro`).
- `src/layouts`: Main layout (`Layout.astro`).
- `public`: Static assets (images, CSS, JS, locales).
